# 契合度测试 💕

一款基于微信小程序的恋爱契合度测试应用，通过趣味问答帮助用户了解自己的恋爱偏好和关系风格。

## 功能特性

- **首页** — 精美启动页，展示应用介绍与协议确认
- **信息填写** — 收集用户基本信息
- **趣味测试** — 多维度心理测评问卷，支持单选与多选
- **结果展示** — 生成个性化匹配报告，包含：
  - 恋爱偏好类型与标签
  - 模型贴合指数（百分比）
  - 详细分析报告（多个维度）
  - 适合/不适合的对象描述
  - 相处建议
  - 关系侧重点雷达图（情绪回应、理性沟通、独立性、承诺意愿、行动倾向）
- **匹配列表** — 探索同频对象
- **联系方式** — 联系与分享入口
- **分享功能** — 支持微信分享测试结果

## 技术栈

- **前端框架**：微信小程序原生开发
- **云开发**：微信云开发（CloudBase）
- **自定义组件**：`navigation-bar` 导航栏组件
- **云函数**：`getOpenid` 获取用户唯一标识

## 项目结构

```
├── app.js / app.json / app.wxss    # 小程序入口与全局配置
├── cloudfunctions/
│   └── getOpenid/                   # 云函数 - 获取用户 openid
├── components/
│   └── navigation-bar/              # 自定义导航栏组件
├── images/                          # 静态图片资源
├── pages/
│   ├── home/                        # 首页
│   ├── info/                        # 信息填写页
│   ├── test/                        # 测试答题页
│   ├── result/                      # 结果展示页
│   ├── match-list/                  # 匹配列表页
│   └── contact/                     # 联系方式页
├── project.config.json              # 项目配置
└── sitemap.json                     # 小程序搜索配置
```

## 使用方式

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 克隆本仓库
3. 在微信开发者工具中导入项目目录
4. 配置你自己的云开发环境 ID（在 `app.js` 中修改 `env` 字段）
5. 上传并部署云函数 `getOpenid`
6. 编译运行
