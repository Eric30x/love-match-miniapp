# 契合度测试 💕

一款基于微信小程序的恋爱契合度测试应用，通过趣味问答帮助用户了解自己的恋爱偏好和关系风格。

## 功能特性

- **首页** — 精美启动页，展示应用介绍与协议确认
- **信息填写** — 收集用户基本信息
- **趣味测试** — 多维度心理测评问卷，支持单选与多选
- **结果展示** — 生成个性化匹配报告，包含：
  - 恋爱偏好类型与标签
  - 模型贴合指数（百分比）
  - 详细分析报告（多个维度）
  - 适合/不适合的对象描述
  - 相处建议
  - 关系侧重点雷达图（情绪回应、理性沟通、独立性、承诺意愿、行动倾向）
- **匹配列表** — 探索同频对象
- **联系方式** — 联系与分享入口
- **分享功能** — 支持微信分享测试结果

## 匹配算法

匹配引擎采用四层递进式架构，逐层筛选并精细化评分：

### 第一层：硬过滤
对不满足基本条件的用户进行快速排除，减少后续计算量。

### 第二层：多维度评分
通过多个维度对候选用户进行综合打分：

- **人格匹配** — 基于心理测评结果的人格特质对比
- **标签匹配** — 恋爱偏好标签的重合度计算
- **自我介绍匹配** — 个人描述文本的语义相似度分析
- **雷达图匹配** — 五维关系侧重点（情绪回应、理性沟通、独立性、承诺意愿、行动倾向）的向量距离

### 第三层：互补匹配
在相似性之外，识别双方的互补特质——某些维度上的差异反而能形成更好的平衡与吸引。

### 第四层：综合加权
将各层得分按权重汇总，输出最终的匹配指数，生成个性化的匹配报告。

## 技术栈

- **前端框架**：微信小程序原生开发

## 使用方式

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 克隆本仓库
3. 在微信开发者工具中导入项目目录
4. 配置你自己的云开发环境 ID（在 `app.js` 中修改 `env` 字段）
5. 上传并部署云函数 `getOpenid`
6. 编译运行
